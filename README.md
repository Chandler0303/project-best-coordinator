# Project Coordinator (项目总指挥)

PM_Back 项目的总指挥 skill，协调所有 skills 的使用，确保开发工作流程顺畅执行。

## 快速使用

```
/project-coordinator [您的任务描述]
```

例如：
- `/project-coordinator 我需要添加一个新功能，应该用哪个 skill？`
- `/project-coordinator 如何修复这个 Bug？`
- `/project-coordinator 如何进行代码审查？`

## 核心功能

- 任务路由：自动识别任务类型，路由到合适的 skill
- 工作流程协调：指导正确的技能使用顺序
- 技能选择：帮助选择最合适的 skill

详见 `SKILL.md`
